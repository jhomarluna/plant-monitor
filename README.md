## Plant Monitor
## An program witten in ArduinoIDE to monitor plants. Uses Arduino UNO R3, Soil Moisture Sensor, DHT11, LEDs, and LCD Monitor
**HARDWARE**
+ Arduino Board
+ LCD1602 Screen
+ 10k ohm Potentiometer
+ 220 ohm Resistor
+ Hook-up wires
+ Breadboard
+ Yellow LED
+ Red LED
+ DHT11
+ Soil Moisture Sensor
